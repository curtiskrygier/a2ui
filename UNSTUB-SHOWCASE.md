**How to hook it up:**
1. Get your Sheet ID from the URL: `docs.google.com/spreadsheets/d/**ID**/edit`
2. Add to `appsscript.json` if not present: `"oauthScopes": ["https://www.googleapis.com/auth/spreadsheets"]`
3. Re-deploy after scope change — GAS will prompt for new permissions on first run
4. Put the ID in `spreadsheet_id` in the atom JSON — the tab named in `sheet` is created automatically
