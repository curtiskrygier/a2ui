## Usage Tips

- **Dark theme triggers**: aviation, space, AI/model comparisons, technical dashboards, cinematic subjects
- **Best atom combos**: `geo_iso_*` + `stat_card` + `terminal_boot` · `decision_tree` + `flip_card` + `quiz_question` · `llm_comparison_table` + `model_card` + `pros_cons_list` · `xp_bar` + `achievement_badge` + `confetti_burst`
- **For recordings**: prompts 1–7 and 47–49 have the most visual impact; prompts 33–36 show interactivity
- **Cmd+Enter** to generate without reaching for the mouse
